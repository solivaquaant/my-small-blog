# Forensics Git 1

- **Description:** Can you find the flag in this disk image?

## 🔎 Solution

Inside the `.git` directory, there are five Git object files:
```
+++++++ r/r 65695:	50f47a5dabfb4397706aa18905df936595a86e
+++++++ r/r 65698:	2340e078686778969b9a555fc722147cf14e5a
+++++++ r/r 65700:	7789af0b300e043ea8f54ea57d6cee352291ae
+++++++ r/r 65708:	825dc642cb6eb9a060e54bf8d69288fbee4904
+++++++ r/r 65709:	b8194539c770a830b8ba089a50778c07072b03
```

Extract these object files from the disk image:
```
icat -o 0001140736 disk.img 65695 > obj1
icat -o 0001140736 disk.img 65698 > obj2
icat -o 0001140736 disk.img 65700 > obj3
icat -o 0001140736 disk.img 65708 > obj4
icat -o 0001140736 disk.img 65709 > obj5
```

To inspect the raw content of Git objects, use `zlib-flate` since Git stores objects compressed with `zlib`. 
The flag appears in the first object.
```
> zlib-flate -uncompress < obj1             
blob 31picoCTF{g17_r3m3mb3r5_d4ddf904} 
```

## 🚩 Flag
> `picoCTF{g17_r3m3mb3r5_d4ddf904} `
