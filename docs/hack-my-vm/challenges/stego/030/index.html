The file provided looked like a QR code at first glance, but something was clearly missing - the 3 positioning squares normally found in the corners weren't there. Without those markers, most scanners wouldn't decode it.

I opened the image in Paint (any basic image editor works) and manually restored the missing alignment squares to match a standard QR structure. Once the pattern was complete, the code became scannable.

After scanning the fixed QR code, the flag was revealed.
