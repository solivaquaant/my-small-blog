# Mission 0x36

## 🔎 Solution
After gaining SSH access as **gloria**, the next objective was to recover the password for the user **alora**.
```
gloria@venus:~$ cat mission.txt 
################
# MISSION 0x36 #
################

## EN ##
User alora likes drawings, that's why she saved her password as ...
```

Inside the directory, there was a file named `image`. Checking its content revealed a pattern resembling a QR code:

```

```
