Based on the challenge name and its description, it is reasonable to assume that this challenge is related to cookies.
On the website, when clicking the **"Saved email addresses"** option, a message appears stating *"You need to be admin"*.

By inspecting the request in Burp Suite, we can see that the `Cookie` header contains the value `ch7=visiteur`.
