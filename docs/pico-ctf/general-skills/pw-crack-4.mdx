This challenge is essentially the same as **PW Crack 3**, but with a longer list of possible passwords. 
The approach remains straightforward - simply add a brute-force routine to iterate through all candidates until the correct one is found.
```python
for pw in pos_pw_list:
    if hash_pw(pw) == correct_pw_hash:
        print(f"Correct password: {pw}")
        flag = str_xor(flag_enc.decode(), pw)

```
