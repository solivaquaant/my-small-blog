# Sleuthkit Intro

- **Description:** Download the disk image and use mmls on it to find the size of the Linux partition.

The challenge provides clear instructions in the description. 
First, we extract the given file and use `mmls` to analyze the partition layout. 
We identify a Linux partition with a length of `0000202752`.
```js
> mmls disk.img      
DOS Partition Table
Offset Sector: 0
Units are in 512-byte sectors

      Slot      Start        End          Length       Description
000:  Meta      0000000000   0000000000   0000000001   Primary Table (#0)
001:  -------   0000000000   0000002047   0000002048   Unallocated
//highlight-next-line
002:  000:000   0000002048   0000204799   0000202752   Linux (0x83)
```
Next, we connect to the server and submit this partition length as the answer - which successfully returns the flag.
